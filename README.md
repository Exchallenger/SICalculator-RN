## SICalculator

#### 주식/코인 물타기,복리,수익률 계산을하기위한 계산기 어플입니다.

#### 현재 플레이스토어에 배포되어있는 상태이며 다운로드수는 100+입니다.

#### 주요기능 
- 물타기 여러 번 하기
- 복리 계산하기
- 수익률 계산 두가지 방법으로 하기

##### 사용된 기술 : react-native, styled-components, expo
